# 💰 SpendWise - Personal Finance Tracker

A modern and responsive Expense Tracker web application built using **HTML, CSS, and JavaScript**. SpendWise helps users manage income and expenses, visualize spending habits, and monitor monthly savings through an intuitive dashboard.

## 🚀 Features

### 💵 Transaction Management

* Add Income and Expense transactions
* Categorize transactions
* Add descriptions and dates
* Delete transactions with confirmation modal

### 📊 Dashboard Overview

* Current Balance
* Total Income
* Total Expenses
* Real-time updates

### 📈 Spending Analytics

* Interactive Doughnut Chart using Chart.js
* Category-wise expense visualization
* Monthly income and expense analysis
* Savings and expense ratio tracking

### 🔍 Search & Filters

* Search transactions by description
* Filter by transaction type
* Filter by category
* Filter by month
* Clear all filters instantly

### 🌙 User Experience

* Dark Mode support
* Responsive design for desktop and mobile
* Toast notifications
* Modern UI with smooth animations

### 💾 Data Persistence

* Local Storage integration
* Transactions remain saved after browser refresh

### 📁 Export Functionality

* Export transaction history to CSV
* Compatible with Microsoft Excel and Google Sheets
